# GreetingApp_2115001051
This Greeting App is a simple ASP.NET application that demonstrates RESTful web services using different HTTP methods. The project is implemented in multiple phases, each adding new functionality to the GreetingController.
# Features
### UC1 - Basic GreetingController
- Implements a GreetingController that returns a JSON response for different HTTP methods.
### UC2 - Service Layer Integration
- Extends GreetingController to use a Service Layer.
- The service layer returns a simple greeting message: "Hello World".
### UC3 - Personalized Greeting
- Ability for the Greeting App to give Greeting message with
- 1. User First Name and LastName or
  2. With just First Name or LastName based on User attributes provides or
  3. Just Hello World.
